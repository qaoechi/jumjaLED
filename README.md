# jumjaLED
소문자 알파벳 ASCII코드를 입력해 점자 출력

## abstract
이 프로젝트는 소문자 알파벳의 ASCII 코드를 2진수로 입력받아, 해당 문자를 점자 형태의 LED 패턴으로 출력하는 하드웨어 설계입니다.

규칙
입력부
변환부
논리게이트

## 규칙
1. 입력 범위
    - 소문자 아스키 코드(97~122)
    - 단순화를 위해 96을 뺀 값을 실제 입력(a = 1, b = 2..., z = 26)
2. 점자 규칙
    - 영어 점자는 10의 주기를 가짐
        + a ~ j: 숫자 점자와 동일(⠁⠃⠉⠙⠑⠋⠛⠓⠊⠚)
        + k ~ t: a ~ j 패턴 + 3번 점 추가
        + u ~ z: a ~ j 패턴 + 3번, 6번 점 추가 (단, w는 제외)
3.  a ~ j를 같은 자릿수로 만들기 위해 추가로 1을 감산하여 스케일을 맞춤

## 입력부
+ 링 카운터 기반 커서 입력
  - 링 카운터가 순환하면서 LED가 켜지는 위치가 현재 입력 비트를 의미(커서 역할)
  - 스위치를 누르면 해당 위치의 비트 값을 1로 저장. 누르지 않으면 유지
  - 스위치 입력 신호를 링 카운터 출력과 AND해 특정 위치만 입력되도록 구성
  - 입력값은 JK플립플롭에 저장
+ 공백 처리
  - 입력이 00000이면 공백이지만 스케일을 위해 1을 감산하기에 별도 처리 필요
  - 입력부의 모든 비트를 OR하고 출력부와 AND하여 공백 표현

## 변환부
+ 아스키 보정
  - 점자 규칙에 맞추기 위해 1 감산
+ 감산된 값을 BCD코드로 변환
  - BCD코드는 10진수를 각 자릿수로 표현
  - BCD코드로 표현된 10의 자릿수로 3,6점 제어
  - BCD코드로 표현된 1의 자릿수로 1,2,4,5점 제어
  - 단, 10의 자릿수가 0010이고 1의 자릿수가 0011 이상이면 1감산 혹은 w 출력
